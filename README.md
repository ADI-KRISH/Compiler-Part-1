# Compiler Part 1

## Overview
The `Compiler Part 1` folder contains the implementation of the first part of a compiler. This project is part of a broader study of compilers and virtual machines.

## Folder Structure

- **codefile.txt**: Input file containing source code to be compiled.
- **CompilationEngine.java**: Java implementation of the compilation engine.
- **Parsein.txt**: Input file for the parser.
- **Parseout.txt**: Output file generated by the parser.
- **tokenedfile.txt**: File containing tokenized output.
- **Tokenizer.java**: Java implementation of the tokenizer.

## Features
- Tokenizes source code into meaningful symbols.
- Parses the tokenized code and generates intermediate output.
- Implements the first stage of a multi-pass compiler.

## How to Use
1. Place the input source code in `codefile.txt`.
2. Run the `Tokenizer.java` program to generate `tokenedfile.txt`.
3. Run the `CompilationEngine.java` program to generate `Parseout.txt`.

## Requirements
- Java Development Kit (JDK) installed.
- IntelliJ IDEA or any Java IDE for running the project.

## Notes
- Ensure the input source code follows the expected syntax.
- The `CompilationEngine.java` file contains methods for parsing and generating intermediate output.
